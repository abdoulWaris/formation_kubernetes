## Questions courtes

### Question 6

Un **Chart** est un ensemble de fichiers permettant de déployer une application. Une **Release** est une instance installée de ce Chart dans un cluster Kubernetes.

---

### Question 7

Le fichier `values.yaml` permet de personnaliser les paramètres d'un Chart, comme le nombre de répliques, l'image Docker ou les ports utilisés.

---

### Question 8

Les templates Helm permettent de générer automatiquement des fichiers YAML en fonction des valeurs fournies, ce qui évite la duplication et facilite la réutilisation.

---

### Question 9

Exemples de réponses :

- simplifie les déploiements ;
- facilite les mises à jour ;
- réduit la duplication des fichiers YAML ;
- permet de partager des applications sous forme de Charts ;
- personnalise facilement les configurations grâce à `values.yaml`.
