# Installation de Minikube

## Présentation

Minikube est une distribution Kubernetes conçue pour apprendre et développer localement.

Elle crée un cluster Kubernetes sur votre ordinateur en utilisant un moteur de virtualisation ou Docker.

## Avantages

- Facile à installer
- Dashboard intégré
- Très populaire pour l'apprentissage
- Compatible Windows, Linux et macOS

## Inconvénients

- Plus gourmand en ressources que Kind
- Principalement destiné au développement local
