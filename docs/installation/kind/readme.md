# Installation de Kind

## Présentation

Kind (Kubernetes IN Docker) est une distribution Kubernetes qui exécute un cluster dans des conteneurs Docker.

Elle est principalement utilisée pour :

- le développement
- les tests
- les laboratoires
- les pipelines CI/CD

## Avantages

- Léger
- Rapide
- Multi-node
- Multi-cluster
- Très utilisé dans les projets GitHub

## Inconvénients

- Pas destiné à la production
