# Kubernetes avec Docker Desktop

## Présentation

Docker Desktop intègre un cluster Kubernetes directement dans son interface.

Il suffit d'activer Kubernetes dans les paramètres pour disposer d'un cluster fonctionnel.

## Avantages

- Installation très simple
- Intégration native avec Docker
- Idéal pour découvrir Kubernetes

## Inconvénients

- Peu configurable
- Un seul cluster
- Moins adapté aux scénarios avancés
