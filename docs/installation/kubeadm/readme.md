# Installation avec kubeadm

## Présentation

kubeadm est l'outil officiel recommandé par Kubernetes pour créer un cluster.

Il permet d'installer un ou plusieurs nœuds afin de construire un environnement proche de la production.

## Avantages

- Installation officielle
- Grande flexibilité
- Convient aux clusters de production

## Inconvénients

- Installation plus complexe
- Nécessite plusieurs étapes de configuration
- Demande une bonne connaissance de Linux et du réseau
